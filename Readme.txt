Учебный проект по анализу данных с тяжёлыми хвостами.
Написан на языке Python, оформлен в Tex.
Датасет состоит из массив данных моделирования случайных графов, полученных с помощью моделей предпочтительного присоединения и кластерного присоединения с разными параметрами для анализа. Данные представляют из себя стобец Excel размером 500 значений. 
Необходимо определить наличие тяжёлого хвоста в данных. 
При помощи таких методов как:
1) QQ-plot.
2) Mean excess function.
3) Эстиматоры Хилла, ratio, momentum, pickands.
4) Доверительный интервал.
Сделаны выводы о наличии тяжёлого хвоста у распределения.

Проект состоит из:
1) Файла Отчёт.pdf с отчётом о проделанной работе.
2) Файла Markovich_Data_Analysis.ipnb с программой.
3) Папки Latex с отчётом о проделанной работе и файлами необходимыми для компеляции отчёта LaTex.
4) Папки Graphics с сохранёнными графиками работы програмы.
5) Папки Materials + Data в которой хранятся предоставленные материалы и данные для обработки.
6) Файла Example_Mikhail.xlsx формата Excel с выборкой данных 7 столбца для обработки.
